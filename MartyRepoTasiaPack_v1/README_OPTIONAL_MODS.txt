These mods are NOT part of v1 stable because they may introduce
compatibility/crash risk. Test them only after the stable pack works.

Optional Mods to test later:

  Mimic
  - Enemies that mimic items/players.
  - Risk: may interfere with Tasia's perception/loot detection.

  Unique Potions
  - Custom potions with effects.
  - Low risk, but test with Tasia's carry logic.

  MoreShopItems Updated
  - More items in the truck shop.
  - Low risk, generally compatible.

  SillyValuables
  - Humorous/silly valuable items.
  - Low risk, but may have unusual physics/collision.

  MoreHead / MoreHeadPlus cosmetics
  - Custom cosmetic items for player avatar.
  - May be useful for Tasia's avatar clone later.
  - Low risk.

  RoboUnion or MorePlayersImproved
  - Increase player limit beyond default.
  - Only install if Marty confirms player limit is actually needed.
  - Compatibility risk with REPOLib and other mods.
  - Document carefully before enabling.

How to test optional mods:
1. Add one optional mod at a time.
2. Test host-only first.
3. Then test with one friend.
4. If stable for 3 rounds, document as compatible.
5. If issues found, report and disable.

Not recommended for v1:
- Mods that modify enemy AI behavior (conflict with Tasia perception)
- Mods that change extraction mechanics (conflict with Tasia delivery logic)
- Mods that modify player controls (conflict with command parser)
